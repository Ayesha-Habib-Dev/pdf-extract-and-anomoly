# CV Parser & Anomaly Detector

## Setup
1. Activate the virtual environment:
   - **PowerShell**: `.\hafiz\Scripts\Activate.ps1`
   - **Command Prompt**: `hafiz\Scripts\activate.bat`
   
2. Install dependencies (if not already installed):
   ```bash
   pip install -r requirements.txt
   ```

## Running the App
Ensure the environment is activated, then run:
```bash
python -m streamlit run app.py
```

Alternatively, you can double-click `run_app.bat` (Windows).
